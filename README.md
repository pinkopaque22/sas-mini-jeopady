SasMiniApp

### installing

to open on your local:

```
git clone
bundle install
use ruby 2.6.3
```

## running the tests

run rspec to run test suite

## built in

* [Rails](http://rubyonrails.org/) - The web framework used
* [jQuery](https://api.jquery.com/) - The JavaScript library used
